## Definition

Converts a given function into a class method.

## Syntax

```py
complex(real, imaginary)
```

## Example 1

Use `complex()` to return the absolute value of `-6.5`:

Here `complex()` will use `0` as the default imaginary value

```py
print(complex(1))
# Output: 6.5
```
